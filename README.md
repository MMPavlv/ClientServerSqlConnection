## ТЗ:
- Функция подключения к БД MS SQL Server;
- Функция, которая возвращает версию MS SQL;
- Функция, которая закрывает подключение к БД.

Написать UI-приложение которое может подключаться к вышеупомянутому сервису и вызывать его методы.

## Пояснения 

### Сервер

Сервер использует ASP.NET. Само задание мне показалось немного странным в том плане, что обычно соединение открывается и закрывается для каждого запроса отдельно. 
Управлением подключением к БД занимается BdService, который регистрирую как синглтон. Он хранит в себе SqlConnection и предоставляет минимальный набор функций для работы с ним: 
- открыть
- закрыть
- получить версию
- получить текущий статус соединения

Обработки исключений нет, фреймворк сам с этим справляется.


### Клиент

Клиент написан на AvaloniaUI с использованием MVVM. Простая одностраничка, разве что попытался сделать немного приличной на вид и использовал гайд от самих разработчиков Авалонии по имитации уведомлений [How to: Show notifications and toasts | Avalonia Docs](https://docs.avaloniaui.net/docs/how-to/notifications-how-to).
Пользователь может указать url до сервера и через кнопочки вызывать соответствующие эндпоинты. Снизу есть имитация лога, туда скидываются тексты ошибок и результаты запросов к серверу. 
Если запрос версии MS SQL Server был успешным, то она выводится в центральном текстовом поле, которое очищается при закрытии соединения.
